CRUD-API

This project was developed using Pyhton, SQLAlchemy, Alembic

API


Fully functional CRUD RESTful API for a social media application

Features
- Create users. - Read users. - User authentication.
- Create posts. - Read all posts. - Read only one post. - Delete your posts. - Edit your posts. - Read only your posts.
- Upvote a post.
- Remove your upvote from a post. - Comment on a post. - Edit your comment from a post. - Remove your comment from a post.
